# ClutchAI: The Autonomous Triage & Shadow-Execution Engine 🚀

**Built for the Vibe2Ship Hackathon**  
*A Next-Generation Proactive Productivity Companion and Multi-Agent Automation System.*

---

## 💡 The Core Problem
Traditional productivity tools and calendar software rely entirely on passive reminders. When high-stakes business or academic deadlines approach, notifications are easily ignored, and the heavily stressed user is still left to do all the heavy lifting manually.

## 🌟 Our Innovation: From Reminder to Autonomous Execution
**ClutchAI** shifts the paradigm from simple tracking to active, autonomous crisis mitigation. When an critical deadline enters the danger zone, the system doesn't just alert the user—it silently acts in the background to build ready-to-submit artifacts, handle scheduling coordination, and prepare communication templates.

---

## 🛠️ Key Product Features

*   **Intelligent Panic Indexing:** Continuously calculates dynamic urgency scores (0–100%) based on real-time deadline proximity and task complexity.
*   **True Browser Web Speech API Integration:** Features a functional natural language Voice Terminal that utilizes browser speech recognition to process spoken triage instructions instantly.
*   **Google Workspace Synergy Grid:** Features a full interactive 4-week calendar matrix highlighting crisis-danger dates with pulsing crimson accents and functional drawer overlays.
*   **Google Tasks Sync Portal:** Pulls active workflow lists into a specialized prioritization board with single-click automation routing.
*   **Google Docs Workspace Replication:** Automatically renders generated solution code, assets, or outlines inside an emulated Google Docs canvas workspace complete with font tools and direct export visual simulations.
*   **Autopilot Governance Panel:** Features an isolated, dedicated workspace separating active operational metrics from strict threshold adjustment rules, slider bounds, and security exclusion keyword strings.
*   **Acoustic Overdrive Escalation:** Implements a hardware-bound browser Web Audio API oscillator synthesis engine to output a procedural electronic multi-tone sound pattern to wake up an unresponsive user during severe crises.

---

## 🧠 Advanced Agent Architecture (Powered by Gemini)
When a high-urgency task drops below the **T-Minus 60 Minutes** threshold, ClutchAI initiates a multi-agent orchestration sequence powered securely by the **Google AI Studio Gemini 1.5 Pro & Flash APIs**:

1.  **The Triage Agent:** Ingests local application status variables, maps the deadline velocity, and sets layout priorities.
2.  **The Scaffold Agent:** Leverages structural JSON schemas to synthesize a production-grade 90%-complete resolution markdown artifact (code patches, financial projections, or slide structures) inside the Google Docs replication window.
3.  **The Logistics Agent:** Automatically blocks out immediate "Deep Work Sprints" around the interactive calendar components and pre-drafts extension or mitigation notification emails in the Gmail pre-crisis queue.
4.  **The 10-Minute Autopilot Override:** If untouched at the T-minus 10-minute line, the system autonomously enters an aggressive neon deployment phase and executes simulated webhook dispatches to submit the task artifact automatically.

## 🚀 Local Installation & Getting Started

To clone, set up, and evaluate this production workspace prototype locally, run the following sequential developer commands in your terminal interface:

### 1. Clone the Code Repository
```bash
git clone <PASTE_YOUR_GITHUB_REPOSITORY_URL_HERE>
cd clutchai-triage-engine
```

### 2. Install Project Dependencies
```bash
npm install
```

### 3. Spin Up Local Development Preview Staging
```bash
npm run dev
```
*The local development server loop will initialize on port 5173 (`http://localhost:5173`).*

### 4. Provide Local API Key Access
1. Open the running browser environment window.
2. Navigate directly to the **Live API Config** navigation tab on the responsive sidebar menu.
3. Securely enter your **Google AI Studio Gemini API Key** into the text input module box.
4. The application connection badge will instantly update to **"Status: Live Connection"**, fully executing real-time multi-agent background routines.
